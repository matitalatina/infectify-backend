# Infectify backend

It provides data about Coronavirus in Italy. The source is [Protezione Civile](https://github.com/pcm-dpc/COVID-19).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
